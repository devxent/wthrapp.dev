# WeatherApp

A simple Weather Application built using Java Swing that allows users to retrieve real-time weather data based on a location they input. The app fetches weather data from an external API and displays it in a user-friendly graphical interface.

## Features

- **Search for Weather**: Users can input a location name to retrieve the current weather data.
- **Weather Information**: The app displays temperature, weather condition, humidity, and wind speed.
- **Dynamic Icons**: The app updates icons dynamically based on the weather condition (e.g., clear, cloudy, rain, snow).
- **Simple and Intuitive GUI**: A user-friendly interface designed with Java Swing.
